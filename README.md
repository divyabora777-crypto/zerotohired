# ZeroToHero 🚀

**ZeroToHero** is a Next-Gen Career DNA Platform built to help students transition into professional roles using AI.

## Features
- **AI Resume Builder**: Storytelling, Form, and Hybrid modes.
- **Career DNA**: Visual radar charts of your professional strengths.
- **20+ AI Tools**: Bullet enhancers, ATS analyzers, and more.
- **Skill Gap Analysis**: Identify and bridge the gap to your dream job.
- **Portfolio Generator**: Instant personal webpage with QR code.

## Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS 4, Framer Motion.
- **AI**: Google Gemini 1.5.
- **Backend**: Supabase (Optional Cloud Sync).

## Getting Started
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
4. Profit!
